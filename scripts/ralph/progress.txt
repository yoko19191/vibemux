## Codebase Patterns
- xterm packages are scoped: `@xterm/xterm`, `@xterm/addon-fit`, `@xterm/addon-webgl` (not the old `xterm` package)
- Tauri 2 icons must be RGBA PNGs (use `magick -size NxN xc:'#color' -type TrueColorAlpha PNG32:file.png`)
- Tauri 2 `app` config does NOT have a `title` field — title goes on individual windows
- Frontend is at `apps/desktop/`, Rust backend at `apps/desktop/src-tauri/`
- Svelte typecheck: `npm run check` from `apps/desktop/`
- Rust typecheck: `cargo check` from `apps/desktop/src-tauri/`
- Rust lib crate name is `vibemux_desktop_lib`
