## Codebase Patterns
- Shelf height offset is referenced in App.svelte as `bottom: Xpx` in `.content-area.has-shelf` and `.nav-indicator.shelf-offset` — update both when changing shelf height
- ShelfCard is a flex column by default; for compact layouts, switch to flex row with `align-items: center`
- Typecheck: run `npm run check` from `apps/desktop/`
- ShelfCard uses `session.name` directly (not via a helper) — use `session.customName ?? session.name` pattern when displaying names
